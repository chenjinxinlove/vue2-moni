# vue2-router-vuex-filmy
vue2-router-vuex-filmy  《实战es2015：深入现代javascript应用开发》书中的前端项目实践，vue2实现。

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
